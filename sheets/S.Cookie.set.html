Void S.Cookie.set(String nom, Variant valeur[, Object options])

Description:

  • Crée un cookie 
    nom
    valant 
    valeur
    et ayant les caractèristiques 
    options
    sur l'ordinateur de l'internaute.

Remarque: 

  • options
    peut contenir 4 propriétés: 
    • expire
      : délai d'expiration du cookie en millisecondes(par défaut dès que le navigateur est quitté)
    • path
      : chemin d'accessibilité du cookie(par défaut l'url qui l'a créé)
    • secure
      : booléen, si mis à 
      true
      , le cookie ne sera envoyé que si le protocole de connexion est sécurisé(SSL etc…).
    • domain
      : nom de domaine(par défaut le serveur sur lequel se trouve la page)

Exemples: 

  • S.Cookie.set('user_id', '234');
//Le cookie user_id a été crée sur l'ordinateur et vaut '234'
  • S.Cookie.set('user_id', '234', { path: '/' });
//Le cookie user_id a été crée sur l'ordinateur et vaut '234'
//Il est accessible partout dans l'arborescence du domaine
