JSailor S.Fn.wrap(String html)

Description:

  • Entoure chaque éléments de la sélection avec le modèle 
    html
    et renvoie l'objet JSailor basé sur les éléments entourants.

Remarques:

  • Les éléments de la sélection seront toujours insérés dans le premier noeud balise de 
    insert
    et le plus loin possible dans l'arborescence de ce dernier.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor.1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').wrap('<i></i>');

        });
        </script>
    </head>
    <body>
        <p>Je suis le premier paragraphe.</p>
    </body>
</html>
  • <html>

        <script type="text/javascript">
        S(function(){

            S('p').wrap('<i>texte1</i>');

        });
        </script>
    </head>
    <body>
        <p>Je suis le premier paragraphe.</p>
    </body>
</html>
  • <html>

JSailor S.Fn.wrap(JSailor insert)

Description:

  • Entoure chaque éléments de la sélection avec les éléments de la collection de 
    insert
    et renvoie un objet JSailor basé sur les éléments entourants.

Remarques: 

  • insert
    comportant des références sur des éléments déjà existants dans 
    document
    alors ces derniers seront déplacés et non clonés lors de l'application sur le premier élément de la sélection. Les applications suivantes seront faites à partir de clones de 
    insert
    .
  • Là aussi, les éléments de la sélection seront toujours insérés dans le premier élément de 
    insert
    et le plus loin possible dans l'arborescence de ce dernier.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor.1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').wrap( S('span') );

        });
        </script>
        <style type="text/css">
        span
        {
            color: red;
        }
        </style>
    </head>
    <body>
        <span>Je suis le premier span.</span>
        <p>Je suis le premier paragraphe.</p>
        <span>Je suis le second span.</span>
    </body>
</html>
  • <html>
    <head>
        <script type="text/javascript" src="jsailor.1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').wrap( S('span') );

        });
        </script>
        <style type="text/css">
        span
        {
            color: red;
        }
        </style>
    </head>
    <body>
        <span>Je suis le premier span.</span>
        <p>Je suis le premier paragraphe.</p>
        <span>Je suis le second span.</span>
    </body>
</html>

JSailor S.Fn.wrap(Element elt)

Description:

  • Entoure chaque éléments de la sélection avec un élément identique à 
    elt
    et renvoie un objet JSailor basé 
    insert
    .

Remarques:

  • Si 
    insert
    est un élément HTML ou un objet JSailor, donc comporte des références sur des éléments déjà existants dans 
    document
    alors ces derniers seront déplacés et non clonés lors de l'application sur le premier élément de la sélection. Les applications suivantes seront faites à partir de clones de 
    insert
    .
  • Les éléments de la sélection seront toujours insérés dans le premier élément de 
    insert
    et le plus loin possible dans l'arborescence de ce dernier.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor.1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').wrap( document.getElementById('replace') );

        });
        </script>
    </head>
    <body>
        <p>Je suis le premier paragraphe.</p>
        <span>Je suis le premier span.</span>
    </body>
</html>
