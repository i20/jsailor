Array S.Array.extend(Array base, Array rajout)

Description:

  • Renvoie 
    base
    étendu avec les items de 
    rajout
    .

Remarques:

  • Les items de 
    rajout
    déjà présents dans 
    base
    seront réécrits.

Exemples: 

  • var test4 = S.Object.extend([true, 1, 'trois'], ['six', 34]);
//test4 = ['six', 34, 'trois']
