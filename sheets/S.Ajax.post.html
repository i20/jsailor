XMLHttpRequest S.Ajax.post(String url, Object data[, Object params][, Function callback])

Description:

  • Effectue une requête HTTP sur l'URL 
    url
    , lui envoie les données contenues dans 
    data
    via la méthode 'POST', en utilisant les données fournies dans 
    params
    et appelle 
    callback
    sur le résultat de la requête en cas de succès.

Remarques: 

  • params
    est optionnel.
  • Il s'agit d'un raccourci qui est identique à: 
    S.Ajax.request({

    url: url,
    method: 'POST',
    data: data,
    success: callback
    //si params est fourni, ses propriétés seront ajoutées à la liste
});

Exemples: 

  • //considérons d'abord un fichier 'action.php' contenant:

<?php

if( $_POST['numid'] )
    echo 'La variable postée "numid" contient: ' . $_POST['numid'];

?>

//considérons maintenant un autre fichier 'post.html' placé dans le même répertoire que 'action.php',
//avec le script suivant:

S.Ajax.post('action.php', { numid: 7890 }, function(text){

    alert(text);
});

//le code ci-dessus affiche un alert contenant 'La variable postée "numid" contient: 7890'
