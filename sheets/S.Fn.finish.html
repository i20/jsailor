JSailor S.Fn.finish(String prop, Boolean toend)

Description:

  • Termine instantanément les animations sur la propriété css 
    prop
    de tous les éléments de la collection et renvoie l'objet JSailor initial.

Remarques:

  • Si le booléen 
    toend
    est passé à 
    true
    alors le style de chaque élément sera mis au style final décrit lors de l'appel à 
    S.Fn.tune()
    et le callback fourni lors de cet appel sera également appliqué si toutes les animations de style sur cet élément ont été amenées à leur valeur finale.
  • Si 
    toend
    est passé à 
    false
    ou n'est pas fourni alors l'animation sur le style sera simplement stoppée, l'élément gardant le style acquis à l'instant précédent. Le callback ne sera pas appelé.
  • Si 
    prop
    n'est pas fourni alors toutes les animations de style seront stoppées par défaut.
  • Il vous est possible de terminer plusieurs animations de style simultanément en utilisant la syntaxe 'prop1 prop2 prop3'.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('#on').click(function(){

                S('p').tune({

                    'font-size': '3em',
                    'color': '#FFA500'

                }, 1000, function(){

                    alert('Animation terminée!');
                });
            });

            S('#off').click(function(){

                S('p').finish();
            });

        });
        </script>
        <style type="text/css">
        p
        {
            color: blue;
        }
        </style>
    </head>
    <body>
        <button id="on">Déclenchez l'effet!</button>
        <button id="off">Finir l'animation.</button>
        <p>Je suis le premier paragraphe de cette page.</p>
        <p>Je suis le second paragraphe.</p>
    </body>
</html>
