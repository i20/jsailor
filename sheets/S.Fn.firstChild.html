JSailor S.Fn.firstChild()

Description:

  • Renvoie un objet JSailor dont la collection est composée des premiers enfants de chaque élément de l'ancienne.

Remarques:

  • L'utilisation de cette fonction est préférable à 
    S.Fn.children().first()
    car plus efficace.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('div').firstChild().css('color', 'red');

        });
        </script>
    </head>
    <body>
        <div>
            <span>Je suis le premier span.</span>
            <p>Je suis le premier paragraphe.</p>
        </div>
        <div>
            <div>
                <span>Je suis le second span.</span>
                <p>Je suis le second paragraphe.</p>
            </div>
            <div>
                <p>Je suis le troisième paragraphe.</p>
                <span>Je suis le troisième span.</span>
            </div>
        </div>
        <p>Je suis le quatrième paragraphe.</p>
    </body>
</html>

JSailor S.Fn.firstChild(String expr)

Description:

  • Renvoie un objet JSailor dont la collection est composée des premiers enfants de chaque élément de l'ancienne qui correspondent au sélecteur 
    expr
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('div').firstChild('span').css('color', 'red');

        });
        </script>
    </head>
    <body>
        <div>
            <span>Je suis le premier span.</span>
            <p>Je suis le premier paragraphe.</p>
        </div>
        <div>
            <div>
                <span>Je suis le second span.</span>
                <p>Je suis le second paragraphe.</p>
            </div>
            <div>
                <p>Je suis le troisième paragraphe.</p>
                <span>Je suis le troisième span.</span>
            </div>
        </div>
        <p>Je suis le quatrième paragraphe.</p>
    </body>
</html>

JSailor S.Fn.firstChild(Function fn)

Description:

  • Renvoie un objet JSailor dont la collection est composée des premiers enfants de chaque élément de l'ancienne qui satisfont le prédicat 
    fn
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('div').firstChild(function(){

                return S(this).tag() === 'SPAN';

            }).css('color', 'red');

        });
        </script>
    </head>
    <body>
        <div>
            <span>Je suis le premier span.</span>
            <p>Je suis le premier paragraphe.</p>
        </div>
        <div>
            <div>
                <span>Je suis le second span.</span>
                <p>Je suis le second paragraphe.</p>
            </div>
            <div>
                <p>Je suis le troisième paragraphe.</p>
                <span>Je suis le troisième span.</span>
            </div>
        </div>
        <p>Je suis le quatrième paragraphe.</p>
    </body>
</html>
