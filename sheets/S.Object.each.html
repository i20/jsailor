Void S.Object.each(Object base, Function fn)

Description:

  • Applique 
    fn
    à chaque propriété de 
    base
    .

Remarques:

  • L'utilisation du mot clé 
    this
    dans le corps de fonction fait référence à la valeur de la propriété(ou item) sur laquelle est appliquée 
    fn
    .
  • fn
    prend 2 paramètres de cette manière, 
    fn(String+Number i, Variant valeur)
    : 
    i
    est la clé(ou item) sur laquelle est appliquée 
    fn
    ; 
    valeur
    est la valeur de la propriété(ou item)(idem 
    this
    ).
  • Si la clé est numérique dans l'objet alors 
    i
    sera un nombre sinon une chaine.
  • Si 
    base
    est un array alors 
    i
    sera toujours un nombre.
  • fn
    peut renvoyer le booléen 
    false
    qui équivaut au mot clé 
    break
    .

Exemples: 

  • S.Object.each({

    prop1: 'un',
    prop2: 2,
    prop3: '3',
    4: 'quatre'

}, function(){

    alert(this);
});
/*affiche 4 alert à la suite de cette façon:

alert('un');
alert(2);
alert('3');
alert('quatre');*/
  • S.Object.each({

    prop1: 'un',
    prop2: 2,
    prop3: '3',
    4: 'quatre'

}, function(i, value){

    //value est identique à this
    alert(i + ': ' + value);
});
/*affiche 4 alert à la suite de cette façon:
alert('prop1: un');
alert('prop2: 2');
alert('prop2: 3');
alert('4: quatre');*/
  • S.Object.each({

    prop1: 'un',
    prop2: 2,
    prop3: '3',
    4: 'quatre'

}, function(i, value){

    //on casse la boucle en arrivant à la propriéte prop3
    if(i === 'prop3'){ return false; }

    alert(i + ': ' + value);
});
/*affiche 2 alert à la suite de cette façon:
alert('prop1: un');
alert('prop2: 2');*/
