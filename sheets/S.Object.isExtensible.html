Boolean S.Object.isExtensible(Object obj)

Description:

  • Renvoie 
    true
    si 
    obj
    est extensible, c'est à dire que l'on peut lui rajouter de nouvelles propriétés.

Remarques:

  • Tous les objets définis par l'utilisateur sont par défaut extensibles.

Exemples: 

  • var obj = {};

alert( S.Object.isExtensible(obj) ? 'extensible' : 'non extensible' );
//affiche un alert contenant 'extensible'

  • alert( S.Object.isExtensible(ActiveXObject) ? 'extensible' : 'non extensible' );
//affiche un alert contenant 'non extensible'(dans le cas où ActiveXObject existe)
