Object S.Fn

Ce sous objet contient toutes les méthodes applicables à n'importe quelle instance JSailor. C'est en réalité le prototype 

S.JSailor.prototype
. La plupart de ces méthodes renvoyant elles mêmes une nouvelle instance JSailor, cela vous permet de "chaîner" vos actions.

Remarques:

  • Il vous est possible de chaîner autant d'actions que vous le souhaitez.
  • Il est impossible de chaîner une action si l'action précédente ne renvoie pas une instance JSailor (ex: 
    S('p').color().append('<p>Para</p>')
    car 
    color()
    renvoie ici une couleur et non une instance JSailor).
  • Il est par contre possible de mettre une action qui renvoie autre chose que du JSailor à la fin d'une chaîne.
  • Les actions seront effectuées dans l'ordre de la chaîne sauf si l'action suit un 
    tune()
    , elle se déclenchera juste après l'appel à ce dernier ne laissant pas le temps à l'animation de se faire d'où l'intérêt du callback (cf: S.Fn.tune).

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.5.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p')
                .html('Paragraphe vert')
                .css('color', 'green');

        });
        </script>
    </head>
    <body>
        <p></p>
    </body>
</html>
    Tester ce code!
