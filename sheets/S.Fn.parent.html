JSailor S.Fn.parent()

Description:

  • Renvoie un objet JSailor dont la collection est composée de tous les parents des éléments de l'ancienne.

Remarques:

  • Le noeud 
    document
    peut faire partie de la collection.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').parent().css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <p>Je suis le troisième paragraphe.</p>
    </body>
</html>

JSailor S.Fn.parent(String expr)

Description:

  • Renvoie un objet JSailor dont la collection est composée des parents des éléments de l'ancienne qui correspondent au sélecteur 
    expr
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').parent('div').css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <p>Je suis le troisième paragraphe.</p>
    </body>
</html>

JSailor S.Fn.parent(Function fn)

Description:

  • Renvoie un objet JSailor dont la collection est composée des parents des éléments de l'ancienne qui satisfont le prédicat 
    fn
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').parent(function(){

                return S(this).tag() === 'DIV';

            }).css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <p>Je suis le troisième paragraphe.</p>
    </body>
</html>
