Void S.String.each(String str, Function fn)

Description:

  • Applique 
    fn
    à chaque caractère de 
    str
    .

Remarques:

  • L'utilisation du mot clé 
    this
    dans le corps de fonction fait référence au caractère courant.
  • La fonction prend 2 paramètres: l'index du caractère courant; le caractère courant(idem 
    this
    ).
  • Si la fonction renvoie 
    false
    la boucle est stoppée(
    break
    ).

Exemples: 

  • S.String.each('Une chaine quelconque', function(i, char){

    alert('position: ' + i + '; caractère: ' + this);
});
//affiche une suite d'alert contenant: 'position: 0; caractère: U', 'position: 1; caractère: n' etc…
