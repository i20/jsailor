Number S.Css.convertSize(Number size, String unitstart, String unitend, Element elt, String prop)

Description:

  • Renvoie la conversion de 
    size
    , exprimée dans l'unité 
    unitstart
    , dans l'unité 
    unitend
    .

Remarque:

  • elt
    est l'élément sur lequel la taille va être appliquée et 
    prop
    la propriété css qui va recevoir cette taille. Ces 2 arguments doivent être fournis si la conversion met en jeux des valeurs relatives('%', 'em', 'ex').
  • La fonction gère les unités: 'px', 'pc', 'pt', 'cm', 'mm', 'in', '%', 'em', 'ex'.
