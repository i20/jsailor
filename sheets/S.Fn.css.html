Variant S.Fn.css(String prop)

Description:

  • Renvoie la valeur de la propriété css 
    prop
    du premier élément de la collection.

Remarques:

  • Les nom des propiétés css peuvent être écrites à la mode javascript(ex: 'fontSize') où à la mode css(ex 'font-size') indifféremment.
  • La propriété de style 
    float
    est accessible via l'écriture 'float' pour tous les navigateurs(ne pas écrire 'cssFloat' ou 'styleFloat').
  • Idem pour 
    opacity
    (ne pas écrire 'filter').

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            alert( S('p').css('background-color') );
            //affiche un alert contenant 'blue' ou sa forme RGB suivant le navigateur

        });
        </script>
        <style>
        p:first-child
        {
            background-color: blue;
        }
        </style>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.css(String prop, Variant value)

Description:

  • Applique la valeur 
    value
    à la propriété css 
    prop
    de chaque élément de la collection et renvoie l'objet JSailor initial.

Remarques:

  • Il vous est possible d'écrire des valeurs sous la forme opérateur/espace/valeur/unité (ex '+ 2px'). La valeur finale sera calculée à partir de l'ancienne valeur pour chaque élément, et sera dans l'unité demandée. Attention à bien mettre un espace car '-10px' n'équivaut pas à '- 10px'! Accessible pour +, -, /, *, %, ^.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').css('background-color', 'orange');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.css(Object css)

Description:

  • Applique le style déterminé par 
    css
    à chaque élément de la collection et renvoie l'objet JSailor initial.

Remarques: 

  • css
    est composé des couples 
    prop_css: valeur_css
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').css({

                'background-color': 'orange',
                'font-size': '2em',
                'border': '2px solid red'
            });

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
