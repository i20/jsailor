Number S.Css.absoluteLeftOf(Element elt)

Description:

  • Renvoie le nombre de pixels qui séparent 
    elt
    du bord gauche de la fenêtre.

Remarque:

  • Cette fonction est utile pour obtenir la position absolue sur les abscisses d'un élément en postion relative(idem 
    left
    pour les éléments en position absolue).
