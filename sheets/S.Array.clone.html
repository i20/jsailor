Array S.Array.clone(Array arr)

Description:

  • Renvoie une copie intégrale de 
    arr
    .

Remarques:

  • La copie peut ainsi être modifiée sans altérer l'array original 
    arr
    .

Exemples: 

  • var modele = ['a', 'b'],
    clone = S.Array.clone(modele),
    pointeur = modele;

clone[0] = 'z';
//clone[0] = 'z', modele[0] = pointeur[0] = 'a'

pointeur[0] = 'e';
//clone[0] = 'z', modele[0] = pointeur[0] = 'e'
