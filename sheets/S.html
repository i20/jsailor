Qu'est ce que JSailor?

JSailor est une biblothèque JavaScript qui vous permet de manipuler les éléments d'un document HTML avec une simplicité étonnante et une syntaxe plus claire qu'avec le code natif. La biblothèque possède également un certain nombre de fonctions prédéfinies assez utiles. Le rédaction du code a été faite dans un soucis de portabilité sur les différents navigateurs. Elle comporte un module assez puissant de gestion des requêtes AJAX(cf S.Ajax) qui vous facilitera la vie et permet également de générer des animations de style(cf S.Fn.tune) dans vos pages alors profitez en!

A qui s'adresse JSailor?

JSailor est destinée à tous ceux qui veulent rajouter du dynamisme et des animations à leurs pages web grâce à JavaScript sans passer des nuits entières à coder. Les développeurs souhaitant rendre leur code plus compréhensible seront aussi ravis d'utiliser la syntaxe simple et intuitive de la librairie. Vous devez néanmoins avoir des connaissances de base en XHTML, CSS et JS.

Comment utiliser JSailor?

Pour pouvoir utiliser la syntaxe JSailor dans votre code JavaScript il vous suffit de télécharger le fichier source de la librairie disponible à l'adresse http://omega.webou.net/jsailor/jsailor-1.1.5.min.js. Une fois le fichier téléchargé, placez le où bon vous semble dans l'arborescence de vos dossiers et n'oubliez pas d'inclure le fichier entre les balises head de votre fichier HTML et de placer votre code, écrit en utilisant la syntaxe JSailor, entre des nouvelles balises de script de cette manière:

<head>
    <script type="text/javascript" src="jsailor-.min.js"></script>
    <script type="text/javascript">

        //Votre code JavaScript utilisant la librairie

    </script>
</head>

Nomenclature du manuel

  • Type0 fn(Type1 argument1, Type2 argument2) signifie: la fonction fn prend deux paramètres, l'un de type Type1 et l'autre de type Type2 et retourne une valeur de type Type0.

  • Un type écrit sous la forme TypeA+TypeB signifie: soit du type TypeA soit du type TypeB.

  • Un argument écrit entre [] signifie que cet argument est facultatif.

  • Vous trouverez dans la doc différents types, ci-dessous une liste exhaustive avec leurs significations respectives:
    • Void: undefined(n'existe pas)
    • Variant: n'importe quel type
    • String: chaîne de caractères
    • Char: chaine de caractères ne comportant qu'un seul caractère
    • JSailor: objet JSailor( type retourné par les expressions S(…) ).
    • Function: une fonction
    • Number: une valeur numérique(nombre, chiffre)
    • Element: noeud HTML de type balise
    • Document: noeud 
      document
    • Collection: collection d'éléments HTML
    • Fragment: fragment HTML(cf documentCreateFragment() )
    • Object: objet javascript, soit littéral(déclaré avec {} ou new Object()), soit instance d'une classe)
    • Array: tableau javascript(déclaré avec [] ou new Array())
    • Event: type Object particulier(propriétés bien définies cf objet event)
    • Multi: rassemble tous les types acceptables pour la création d'une instance JSailor soient:
      • Array(ne contenant que des éléments HTML)
      • String(sous forme d'une expression de sélection d'éléments)
      • Function
      • Element
      • Document
      • Collection
