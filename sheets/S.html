What is JSailor?

JSailor is a JavaScript framework that enables you to easily manipulate elements in a HTML document and write scripts in a more concise way. The framework contains a lot of useful methods to help you setting your web application without spending hours on tricky things. The framework is meant to be cross-browser and therefore you don't have to worry about compatibility when writing code with it. One of the great assets of the framework is its AJAX module (cf S.Ajax) especially designed for web applications that relies a lot on AJAX loads.

What is the difference between JSailor and jQuery?

At a first glance, both are basically the same except that JSailor is maintained only by me. Indeed JSailor is actually a personal project whose aim is to try implementing the very good idea of jQuery in a slightly different manner.

How to use?

To get it working, you just need to download the framework available here http://omega.webou.net/jsailor/jsailor-1.1.7.min.js and then call it in the head of your webpages before your own scripts like in the example below:

<head>
    <script type="text/javascript" src="jsailor-.min.js"></script>
    <script type="text/javascript">

    // Put your magic here!

    </script>
</head>

The framework is wrapped in a simple function on which are attached modules (cf tree on the left). Modules contain functions that you can use at your convenience depending on what you want to do. A simple example:

var myString = S.String.stripSlashes('I\\'m a string');
// myString = I'm a string

Now you know how to find the functions, you want to handle HTML elements. There is no easier way to do it, just pass the selector corresponding to the wanted element to the framework function and then you can call the framework's methods on the collection retrieved by the selector like this:

S('div.foo').click(function(){

    S(this).tune({

        'background-color': 'green',
        'font-size': '3em'

    }, 1500, 'bounce-out');
});

// This short code add an on click listener to all divs with the class foo. When clicked the divs will be animated.

Click Me!

Notice that passing a selector to S() is basically the same as passing it to S.JSailor(). For more details on how to select elements from the document see S.JSailor, for information about the methods you can call on a set of elements see S.Fn.
