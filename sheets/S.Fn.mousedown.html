JSailor S.Fn.mousedown(Function callback)

Description:

  • Raccourci pour 
    S.Fn.bind('mousedown', callback)
    (cf S.Fn.bind).

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').mousedown(function(){
            
                alert('Evénement détecté!');
            });

        });
        </script>
    </head>
    <body>
        <p>Cliquez ici!</p>
    </body>
</html>
    Tester ce code!
