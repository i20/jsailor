Object S.Filter

Sous objet de la librairie contenant les filtres de sélection d'éléments couramment appelés pseudo-classes tels que ':nth-child()', ':input' etc…

Cet objet a été rendu accessible dans le but de faciliter son extension par les programmeurs tiers, il suffit pour cela de lui rajouter une propriété du nom du filtre écrite à la mode JavaScript (camelCase) dont la valeur est une fonction avec paramètre si la pseudo-classe en nécessite (cf :nth-child) ou pas (cf :input). Cette fonction devra être un prédicat sur élément (ie renvoyer true si l'élément correspond à la pseudo-classe, false sinon). Dans le corps de la fonction le mot clé this désigne l'élément testé.

Exemples: 

  • //un exemple bidon mais qui vaut mieux qu'un long discours
S.Filter.isDiv = function(){

    return this.tagName === 'DIV';
};

//utilisable de cette manière par la suite
S(':is-div')…
//notons que ceci est strictement équivalent à S('div')
