Void S.Event.unbind(Element elt, String type, Function fn)

Description:

  • Détache le gestionnaire d'événement 
    ontype = fn
    de 
    elt
    .

Remarques: 

  • type
    correspond au type d'événement non préfixé par 'on'.
  • Les autres gestionnaires d'événement de même type ne seront pas touchés.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            var paras = document.getElementsByTagName('p'),
                fn = function(){

                    alert( this.innerHTML );
                };

            //On attache tout d'abord un événement au premier paragraphe:
            S.Event.bind(paras[0], 'click', fn);

            //On détache ensuite l'événement attaché précédemment:
            S.Event.unbind(paras[0], 'click', fn);
        });
        </script>
    </head>
    <body>
        <div>
            <p>Cliquez sur moi!</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
