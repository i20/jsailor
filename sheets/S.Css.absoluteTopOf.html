Number S.Css.absoluteTopOf(Element elt)

Description:

  • Renvoie le nombre de pixels qui séparent 
    elt
    du bord haut de la fenêtre.

Remarque:

  • Cette fonction est utile pour obtenir la position absolue sur les ordonnées d'un élément en postion relative(idem 
    top
    pour les éléments en position absolue).
