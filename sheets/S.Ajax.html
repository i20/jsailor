Object S.Ajax

Sub object of the framework containing all the functions relative to HTTP requests through JavaScript aka AJAX(Asynchronous JavaScript And XML).

Overview

Methods :

  • xhr() : returns the proper request object according to the user's browser, useful when you want to manually handle requests.
  • request() : performs a complete request with all provided parameters.
  • get() : shortcut for request() with method GET.
  • post() : shortcut for request() with method POST.

Properties :

  • settings : global configuration shared between all requests.
