Boolean S.Test.isFragment(Variant valeur)

Description:

  • Renvoie 
    true
    si 
    valeur
    est un fragment HTML, 
    false
    sinon.

Exemples:

  • var valeur = document;
alert(S.Test.isFragment(valeur) ? 'fragment HTML' : 'autre type');
//affiche un alert contenant 'autre type'
  • var valeur = document.documentFragment();
alert(S.Test.isFragment(valeur) ? 'fragment HTML' : 'autre type');
//affiche un alert contenant 'fragment HTML'
