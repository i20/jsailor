JSailor S.Fn.extend(Object obj)

Description:

  • Etend le prototype de l'objet JSailor avec les propriétés de 
    obj
    .

Remarques:

  • Fonctionne identiquement à S.Object.extend appliquée sur l'objet parent.
  • Cette méthode est surtout utile pour les développeurs souhaitant étendre JSailor en y ajoutant des méthodes.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S.Fn.extend({

            maMethode: function(info){

                alert( this[info]() );
            }
        });

        S(function(){

            S('p').click(function(){

                S(this).maMethode('html');
            });

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
