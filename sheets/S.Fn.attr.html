Variant S.Fn.attr(String attr)

Description:

  • Renvoie la valeur de l'attribut 
    attr
    du premier élément de la collection.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            alert( S('p').attr('newattr') );
            //affiche un alert contenant 'something'

        });
        </script>
    </head>
    <body>
        <div>
            <p newattr="something">Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.attr(String attr, Variant value)

Description:

  • Applique la valeur 
    value
    à l'attribut 
    attr
    de chaque élément de la collection et renvoie l'objet JSailor initial.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').attr('class', 'custom');
            S('.custom').css('color', 'red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.attr(Object attrs)

Description:

  • Applique les valeurs correspondantes aux propriétés de 
    attrs
    aux attributs de chaque élément de la collection et renvoie l'objet JSailor initial.

Remarques: 

  • attrs
    est composé des couples 
    nom_attr: valeur_attr
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').prop({

                'class': 'custom',
                'newattr': 'something'
            });

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
