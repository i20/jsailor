JSailor S.Fn.fire(String type, Event event)

Description:

  • Déclenche l'événement 
    type
    ayant les caractèristiques 
    event
    (cf objet event) sur tous les éléments de la collection et renvoie l'objet JSailor initial.

Remarques:

  • type
    correspond au type d'événement non préfixé par 'on'.
  • Il vous est possible de déclencher plusieurs événements simultanément en utilisant la syntaxe 'type1 type2 type3' pour 
    type
    .
