Boolean S.Test.isset(Variant valeur)

Description:

  • Renvoie 
    true
    si 
    valeur
    contient quelque chose, 
    false
    sinon.

Remarques:

  • Si valeur contient 
    null
    la méthode renvoie 
    true
    .

Exemples:

  • var valeur = 'something';
alert(S.Test.isset(valeur) ? 'utilisée' : 'vide');
//affiche un alert contenant 'utilisée'
  • var valeur = null;
alert(S.Test.isset(valeur) ? 'utilisée' : 'vide');
//affiche un alert contenant 'utilisée'
  • var valeur;
alert(S.Test.isset(valeur) ? 'utilisée' : 'vide');
//affiche un alert contenant 'vide'
