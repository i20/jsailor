JSailor new S.JSailor(String expr)

Description:

  • Renvoie une nouvelle instance JSailor basée sur les éléments décrits par le sélecteur 
    expr
    . Pour connaître les expressions possibles, suivez le lien sélecteurs.

Exemples: 

  • var foos = S('.foo');
//foos est un objet JSailor basé sur les éléments HTML ayant l'attribut CLASS égal à foo

JSailor new S.JSailor(Element elt)

Description:

  • Renvoie une nouvelle instance JSailor basée sur l'élément 
    elt
    .

Exemples: 

  • var foo = S(document.getElementById('#foo'));
//foo est un objet JSailor basé sur l'élément identifié par l'ID foo

JSailor new S.JSailor(Collection coll)

Description:

  • Renvoie une nouvelle instance JSailor basée sur les éléments de la collection HTML 
    coll
    .

Exemples: 

  • var spans = S(document.getElementByTagName('span'));
//spans est un objet JSailor basé sur les éléments SPAN du document

JSailor new S.JSailor(Array arr)

Description:

  • Renvoie une nouvelle instance JSailor basée sur les éléments contenus dans l'array 
    arr
    .

Exemples: 

  • var collec = S([elt1, elt2, elt3]);
/*elt1-3 sont supposés être des éléments HTML
collec est alors un objet JSailor basé sur ces 3 éléments*/

JSailor new S.JSailor(Function fn)

Description:

  • Exécute la fonction 
    fn
    lorsque le DOM est prêt à être manipulé. Plus efficace que l'événement 
    onload
    de 
    window
    qui ne se déclenche qu'après le chargement des images.

Exemples: 

  • S(function(){

    //votre code ne devant être éxécuté qu'après le chargement de la page
});

Remarques: 

  • S()
    est identique à 
    new S.JSailor()
    .
  • Si aucun argument n'est fourni au constructeur ou que sa valeur est "fausse"(
    undefined
    , 
    null
    , 
    false
    , 0, '' etc…) alors l'instance démarre sans éléments sélectionnés(utile pour rajouter des éléments HTML avec une boucle).
  • La collection d'un objet JSailor ne peut contenir que des éléments HTML et/ou le noeud 
    document
    , jamais de noeuds texte.
