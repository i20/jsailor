Number S.String.indexOfEndingBracket(Number index, String chaine)

Description:

  • Renvoie la position dans 
    chaine
    de la parenthèse opposée à celle située à la position 
    index
    ou -1 si la parenthèse fermante n'a pas été trouvée.

Exemples:

  • var chaine = 'Une chaine(suite de caractères(symbole de l\'alphabet))';
alert( S.String.indexOfEndingBracket(10, chaine) );
//affiche un alert contenant 54
