Boolean S.Math.between(Number n, Number a, Number b)

Description:

  • Teste si 
    n
    est dans l'intervalle 
    [a,b[
    , renvoie 
    true
    si c'est le cas, 
    false
    sinon.

Exemples: 

  • document.write( S.Math.between(5.8, 4, 7) );
//affiche true

  • document.write( S.Math.between(6, 4, 6) );
//affiche false

  • document.write( S.Math.between(4, 4, 6) );
//affiche true

  • document.write( S.Math.between(7, 4, 6) );
//affiche false
