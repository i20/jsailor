JSailor S.Fn.tune(Object css, Number delay, [Function callback])

Description:

  • Déclenche une animation progressive pour amener chaque élément de la collection au style défini par 
    css
    dans l'intervalle de temps 
    delay
    et renvoie l'objet JSailor initial.

Remarques: 

  • css
    est un objet construit comme pour 
    S.Fn.css({…})
    , il est donc possible d'utiliser les valeurs relatives(ex '+ 12px') et également les notations javascript et/ou css pour les noms des propriétés(cf S.Fn.css).
  • La méthode ne peut prendre en compte que des propriétés css qui se prêtent à une variation progressive comme celles ayant des valeurs avec des unités('height', 'font-size', 'azimut', etc…), celles qui sont des ratios('opacity', etc…), celle qui sont des entiers('z-index', 'zoom', etc…) et également les couleurs!('rgb', 'hsl', etc…).
  • delay
    correspond à une durée en millisecondes.
  • callback
    est une fonction optionnelle qui sera appliquée à chaque élément de la collection une fois l'animation complètement terminée sur celui-ci.

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('button').click(function(){

                S('p').tune({

                    'font-size': '3em',
                    'color': '#FFA500'

                }, 1000, function(){

                    alert('Animation terminée!');
                });
            });

        });
        </script>
        <style type="text/css">
        p
        {
            color: blue;
        }
        </style>
    </head>
    <body>
        <p>Je suis le premier paragraphe de cette page.</p>
        <p>Je suis le second paragraphe.</p>
        <button>Déclenchez l'effet!</button>
    </body>
</html>
