Number S.Css.pxTopct(Number size, Element elt, String prop)

Description:

  • Renvoie la conversion de 
    size
    , exprimée en pixels('px'), en pourcents('%').

Remarque:

  • elt
    est l'élément sur lequel la taille va être appliquée et 
    prop
    la propriété css qui va recevoir cette taille. Ces 2 arguments doivent être fournis car la conversion met en jeux des valeurs relatives('%', 'em', 'ex').
