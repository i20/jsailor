Object S.Css.toRGBa(String color)

Description:

  • Renvoie un objet(
    { red: valeur_rouge, green: valeur_vert, blue: valeur_bleu, opac: valeur_opacite }
    ) contenant la valeur de 
    color
    sur chaque canal.

Remarque:

  • Le canal opacité n'est utile que pour les couleurs CSS3 
    rgba
    et 
    hsla
    .
  • La fonction gère les couleurs: '#FFA500', '#FFF', 'red', 'rgb()', 'hsl()', 'rgba()', 'hsla()'.
