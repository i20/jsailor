Array S.Array.unsetValue(Array tableau, Variant valeur)

Description:

  • Renvoie 
    tableau
    où toutes les occurrences de 
    valeur
    ont été supprimées.

Exemples:

  • var tableau1 = ['moche', 'beau', 'truc', 'moche', 'moche'],
    tableau2 = S.Array.unsetValue(tableau1, 'moche');
//tableau2 = ['beau', 'truc']
