String S.Fn.toString()

Description:

  • Renvoie la chaine de caractères '[object JSailor]' représentant l'objet de type JSailor.

Remarques:

  • Il s'agit de la méthode dite magique associée à tout instance javascript et qui permet de détecter directement le type.
  • Pour de plus amples informations sur la méthode, consultez une documentation javascript.

Exemples: 

  • document.write( S('.foo').toString() );
//affiche '[object JSailor]'

  • document.write( S('.foo') );
//affiche '[object JSailor]'
