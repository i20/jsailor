Void S.Ajax.get(String url[, Object params][, Function callback])

Description:

  • Effectue une requête HTTP sur l'URL 
    url
    en utilisant la méthode 'GET', les données fournies dans 
    params
    et appelle 
    callback
    sur le résultat de la requête en cas de succès.

Remarques: 

  • params
    est optionnel.
  • Il s'agit d'un raccourci qui est identique à: 
    S.Ajax.request({

    url: url,
    method: 'GET',
    success: callback
    //si params est fourni, ses propriétés seront ajoutées à la liste
});

Exemples: 

  • //considérons d'abord un fichier 'data.json' contenant:

{
    "prop1": "val1",
    "prop2": "val2"
}

//considérons maintenant un autre fichier 'get.html' placé dans le même répertoire que 'data.json',
//avec le script suivant:

S.Ajax.get('data.json', { response: 'auto' }, function(json){

    alert(json.prop1);
});

//le code ci-dessus affiche un alert contenant 'val1'
