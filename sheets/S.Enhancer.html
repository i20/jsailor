Object S.Enhancer

Sous objet de la librairie permettant la configuration des extensions HTML. Les extensions HTML sont des scripts ajoutant des comportements spécifiques aux éléments HTML de base.

Remarques:

  • Ces scripts sont lancés de manière asynchrone (non bloquante) dès que le document est manipulable.
  • Lorsque toutes les extensions activées ont été appliquées, un événement JSailorDOMExtended est déclenché.
  • Par défaut toutes les extensions sont désactivées.

Exemples: 

  • S.Enhancer.xxx.on = true;
//active l'extension xxx

S.Enhancer.xxx.on = false;
//désactive l'extension xxx
