JSailor S.Fn.ancestors()

Description:

  • Renvoie un objet JSailor dont la collection est composée par les tous ancêtres des éléments de l'ancienne.

Remarques:

  • Le noeud 
    document
    fera partie de la collection.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').ancestors().each(function(){

                //car le noeud document n'est pas une balise
                if( S.Test.isElement(this) ){

                    alert( S(this).tag() );
                }
            });

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.ancestors(String expr)

Description:

  • Renvoie un objet JSailor dont la collection est composée par les ancêtres des éléments de l'ancienne qui correspondent au sélecteur 
    expr
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').ancestors('div').css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.ancestors(Function fn)

Description:

  • Renvoie un objet JSailor dont la collection est composée par les ancêtres des éléments de l'ancienne qui satisfont le prédicat 
    fn
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').ancestors(function(){

                return S(this).tag() === 'DIV';

            }).css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
