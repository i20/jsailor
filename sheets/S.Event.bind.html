Void S.Event.bind(Element elt, String type, Function fn)

Description:

  • Attache le gestionnaire d'événement 
    ontype = fn
    à 
    elt
    .

Remarques: 

  • type
    correspond au type d'événement non préfixé par 'on'.
  • Les anciens gestionnaires d'événement du même type attachés à 
    elt
    ne seront pas écrasés.
  • fn
    prend en paramètre l'objet événement.
  • fn
    peut retourner un booléen: 
    false
    , bloque l'action par défaut de l'événement(idem 
    event.preventDefault()
    ); 
    true
    , stoppe la propagation de l'événement(idem 
    event.stopPropagation()
    ).
  • fn
    peut retourner un booléen: 
    false
    , bloque l'action par défaut de l'événement(idem 
    event.preventDefault()
    ); 
    true
    , stoppe la propagation de l'événement(idem 
    event.stopPropagation()
    ).

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            var paras = document.getElementsByTagName('p');

            S.Event.bind(paras[0], 'click', function(){

                alert( this.innerHTML );
            });

            S.Event.bind(paras[1], 'mouseover', function(event){

                alert( event.pageX );
            });
        });
        </script>
    </head>
    <body>
        <div>
            <p>Cliquez sur moi!</p>
            <p>Passez la souris sur moi!</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
