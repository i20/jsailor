Object S.Object.clone(Object obj)

Description:

  • Renvoie une copie intégrale de 
    obj
    .

Remarques:

  • La copie peut ainsi être modifiée sans altérer l'objet original 
    obj
    .

Exemples:

  • var modele = {

    prop1: 'a',
    prop2: 'b'    
},
clone = S.Object.clone(modele),
pointeur = modele;

clone.prop1 = 'z';
//clone.prop1 = 'z', modele.prop1 = pointeur.prop1 = 'a'

pointeur.prop1 = 'e';
//clone.prop1 = 'z', modele.prop1 = pointeur.prop1 = 'e'
