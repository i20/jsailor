JSailor S.Fn.unbind(String type, Function fn)

Description:

  • Détache le gestionnaire d'événement 
    ontype = fn
    de tous les éléments de la collection et renvoie l'objet JSailor initial.

Remarques:

  • type
    correspond au type d'événement non préfixé par 'on'.
  • Les autres gestionnaires d'événement de même type ne seront pas touchés.
  • Il vous est possible de détacher plusieurs gestionnaires d'événements simultanément en utilisant la syntaxe 'type1 type2 type3' pour 
    type
    .
