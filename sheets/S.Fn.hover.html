JSailor S.Fn.hover(Function fnover, Function fnout)

Description:

  • Applique 
    fnover
    et 
    fnout
    à tous les éléments de la sélection, respectivement lorsque la souris et sur et en dehors de l'élément.

Exemples: 

  • S('#hover_fn').hover(function(){

    S(this).html('La souris est sur moi!');

}, function(){

    S(this).html('La souris est ailleurs.');
});
//passez la souris sur le texte ci-dessous pour tester ce code


JSailor S.Fn.hover(Object css)

Description:

  • Accomplit la même action que la pseudo-classe 
    :hover
    de css, le style de l'élément sur lequel est la souris étant 
    css
    .

Exemples: 

  • S('#hover_css').hover({

    'color': 'red',
    'font-size': '2em'
});
