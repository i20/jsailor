JSailor S.Fn.each(Function fn)

Description:

  • Applique 
    fn
    à tous les éléments de la collection et renvoie l'objet JSailor initial.

Remarques:

  • Fonctionne identiquement à S.Object.each appliquée sur la collection.
  • Dans 
    fn
    , 
    this
    fait référence à l'élément courant.

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('p').each(function(){

                var old_html = S(this).html();

                S(this).html('Je suis le ' + old_html + ' paragraphe de cette page!');
            });

        });
        </script>
    </head>
    <body>
        <div>
            <p>premier</p>
            <p>second</p>
        </div>
        <div>
            <p>troisième</p>
        </div>
    </body>
</html>
