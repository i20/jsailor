Boolean S.Filter.css(String styles)

Description:

  • Teste si l'élément courant a les style défini par 
    styles
    .

Remarques:

  • Pour tester plusieurs attributs de styles, il suffit de séparer chaque bloc nom/valeur par une virgule.

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.5.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S(':css(display=inline,color=red)').each(function(){
            
                alert(S(this) + ' contenant: ' + S(this).html());
            });

        });
        </script>
    </head>
    <body>

    </body>
</html>
    Tester ce code!
