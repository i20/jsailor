String S.Test.toString(Variant valeur)

Description:

  • Renvoie une chaîne de caractère représentant le constructeur de 
    valeur
    .

Remarques:

  • La valeur de retour dépend du navigateur.
  • Cette fonction est en faite la méthode 
    toString()
    de l'objet Object(idem 
    Object.prototype.toString
    ).
  • Certaines valeurs de retour sont communes à tous les navigateurs comme:
    • '[object Function]' si 
      valeur
      est une fonction
    • '[object Array]' si 
      valeur
      est un array
    • '[object String]' si 
      valeur
      est une chaîne de caractères
    • '[object Number]' si 
      valeur
      est un nombre
    • '[object Boolean]' si 
      valeur
      est un booléen
    • '[object Text]' si 
      valeur
      est un noeud texte
    • '[object Object]' si 
      valeur
      est un objet

Exemples:

  • var valeur = ['un', 'deux'];
alert(S.Test.toString(valeur));
//affiche un alert contenant '[object Array]'
  • var valeur = document;
alert(S.Test.toString(valeur));
/*affiche un alert contenant '[object HTMLDocument]' si le navigateur est Firefox ou Safari(voire Opera),
'[object]' si vous utilisez Internet Explorer*/
