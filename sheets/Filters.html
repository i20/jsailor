Pseudos classes de filtrage sans argument.

  • ':odd': sélectionne les éléments de rang impair.
  • ':even': sélectionne les éléments de rang pair.
  • ':input': sélectionne les éléments de type BUTTON, TEXTAREA, SELECT et INPUT.
  • ':button': sélectionne les éléments de type BUTTON et INPUT button.
  • ':text': sélectionne les éléments de type TEXTAREA et INPUT text.
  • ':radio': sélectionne les éléments de type INPUT radio.
  • ':reset': sélectionne les éléments de type INPUT reset.
  • ':checkbox': sélectionne les éléments de type INPUT checkbox.
  • ':file': sélectionne les éléments de type INPUT file.
  • ':password': sélectionne les éléments de type INPUT password.
  • ':submit': sélectionne les éléments de type INPUT submit.
  • ':image': sélectionne les éléments de type INPUT image.
  • ':checked': sélectionne les éléments de type RADIO et CHECKBOX cochés.
  • ':empty': sélectionne les éléments n'ayant pas d'enfants(balise ou texte).
  • ':parent': sélectionne les éléments ayant au moins un enfant(balise ou texte).
  • ':tuned': sélectionne les éléments en cours d'animation via la S.Fn.tune().
  • ':first-child': sélectionne les éléments qui sont le premier enfant de leur parent.
  • ':last-child': sélectionne les éléments qui sont le dernier enfant de leur parent.
  • ':only-child': sélectionne les éléments qui sont l'unique enfant de leur parent.
  • ':first-of-type': sélectionne les éléments qui sont le premier enfant de ce type de leur parent.
  • ':last-of-type': sélectionne les éléments qui sont le dernier enfant de ce type de leur parent.
  • ':only-of-type': sélectionne les éléments qui sont l'unique enfant de ce type de leur parent.
