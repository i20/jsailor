JSailor S.Fn.clone()

Description:

  • Renvoie un objet JSailor basé sur les clones des éléments du précédent.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('div').append( S('.clones').clone() );

        });
        </script>
        <style type="text/css">
        #clone_parent
        {
            color: red;
        }
        </style>
    </head>
    <body>
        <p class="clones">Paragraphe 1</p>
        <div>Je suis le parent des clones.</div>
        <span class="clones">Span 1</span>
    </body>
</html>
