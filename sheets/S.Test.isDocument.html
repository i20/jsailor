Boolean S.Test.isDocument(Variant valeur)

Description:

  • Renvoie 
    true
    si 
    valeur
    est le noeud document, 
    false
    sinon.

Exemples:

  • var valeur = S('#foo').range[0];
alert(S.Test.isDocument(valeur) ? 'noeud document' : 'autre type');
//affiche un alert contenant 'autre type'
  • var valeur = document;
alert(S.Test.isDocument(valeur) ? 'noeud document' : 'autre type');
//affiche un alert contenant 'noeud document'
