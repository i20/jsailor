JSailor S.Fn.descendants()

Description:

  • Renvoie un objet JSailor dont la collection contient tous les éléments descendants de ceux de l'ancienne.

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('body').descendants().css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.descendants(String expr)

Description:

  • Renvoie un objet JSailor dont la collection est composée par les descendants des éléments de l'ancienne qui correspondent au sélecteur 
    expr
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('body').descendants('p').css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>

JSailor S.Fn.descendants(Function fn)

Description:

  • Renvoie un objet JSailor dont la collection est composée par les descendants des éléments de l'ancienne qui satisfont le prédicat 
    fn
    .

Exemples: 

  • <html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            S('body').descendants(function(){

                return S(this).tag() === 'P';

            }).css('border', '2px solid red');

        });
        </script>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe de cette page.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <div>
            <p>Je suis le troisième paragraphe.</p>
        </div>
    </body>
</html>
