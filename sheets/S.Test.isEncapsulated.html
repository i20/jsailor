Boolean S.Test.isEncapsulated(Char+Number argument, String chaine)

Description:

  • Renvoie 
    true
    si 
    argument
    est entre parenthèses dans 
    chaine
    , 
    false
    sinon.

Remarques:

  • Si 
    argument
    est un caractère alors la méthode testera l'encapsulation de la première occurence de 
    argument
    dans 
    chaine
    .
  • Si 
    argument
    est un nombre alors la méthode testera l'encapsulation du caractère situé à la position 
    argument
    dans 
    chaine
    .
  • Un caractère situé après une parenthèse ouvrante ne se refermant pas est considéré comme encapsulé.

Exemples:

  • alert(S.Test.isEncapsulated('s', 'ami(s)') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'encapsulé'
  • alert(S.Test.isEncapsulated('e', 'amie(s)') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'libre'
  • alert(S.Test.isEncapsulated(4, 'ami(s)') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'encapsulé'
  • alert(S.Test.isEncapsulated(4, 'ami(s') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'encapsulé'
  • alert(S.Test.isEncapsulated('s', 'ami)(s)') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'encapsulé'
  • alert(S.Test.isEncapsulated('s', 'super ami(s)') ? 'encapsulé' : 'libre');
//affiche un alert contenant 'libre'
