Boolean S.Test.isElement(Variant valeur)

Description:

  • Renvoie 
    true
    si 
    valeur
    est un élément HTML, 
    false
    sinon.

Exemples:

  • var valeur = S('#foo').range[0];
alert(S.Test.isElement(valeur) ? 'élément HTML' : 'autre type');
//affiche un alert contenant 'élément HTML'
  • var valeur = document;
alert(S.Test.isElement(valeur) ? 'élément HTML' : 'autre type');
//affiche un alert contenant 'autre type'
