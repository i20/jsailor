Object S.Css.getStyleOf(Element elt)

Description:

  • Renvoie un objet contenant le style de 
    elt
    sous la forme 
    propriete_css: valeur
    .

Remarque:

  • Les propriétés css sont accessibles par leur nom à la mode JavaScript(
    fontSize
    au lieu de 
    font-size
    ).

Exemples: 

  • <!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jsailor-1.1.4.min.js"></script>
        <script type="text/javascript">
        S(function(){

            var style = S.Css.getStyleOf( document.getElementsByTagName('p')[0] );

            alert( style.color );
            //affiche un alert contenant 'red' ou sa valeur RGB

        });
        </script>
        <style type="text/css">
        body
        {
            color: red;
        }
        </style>
    </head>
    <body>
        <div>
            <p>Je suis le premier paragraphe.</p>
            <p>Je suis le second paragraphe.</p>
        </div>
        <p>Je suis le troisième paragraphe.</p>
    </body>
</html>
