Objet événement(Event object)

Il s'agit d'un objet littéral JavasScript avec des propriétés particulières qui permettent de caractèriser un événement(click, mouseover etc…) avec des informations telles que la position de la souris, la touche appuyée etc… Il trouve toute son utilité dans le cas où l'événement a été déclenché artificiellement. Par exemple l'événement click déclenché au survol d'un lien, ce dernier n'étant pas issu d'un véritable clic, l'événement n'est pas renseigné sur le numéro du bouton qui aurait effectué le clic mais via l'objet événement vous pouvez lui donner l'information tout de même.

L'objet événement comprend un grand nombre de propriétés possibles en fonction du type d'événement déclenché, en voici une liste exhaustive avec leur signification et valeurs possibles:

  • Propriétés générales:
    • String 
      type
      : type de l'événement non préfixé par 'on'.
    • Element 
      target
      : élément responsable du déclenchement de l'événement.
    • Element 
      srcElement
      : Idem 
      target
      . Uniquement pour IE.
    • Element 
      currentTarget
      : élément atteint par l'événement à un instant donné. Non compatible IE.
    • Boolean 
      bubbles
      : 
      true
      si l'événement peut bouillonner. Non compatible IE.
    • Boolean 
      cancelable
      : 
      true
      si l'événement peut être supprimé. Non compatible IE.
    • Function 
      stopPropagation
      : stoppe la propagation de l'événement.
    • Boolean 
      cancelBubble
      : 
      true
      = 
      stopPropagation()
      . IE uniquement.
    • Function 
      preventDefault
      : empêche l'action par défaut d'être effectuée.
    • Boolean 
      returnValue
      : 
      false
      = 
      preventDefault()
      . IE uniquement.
    • Number 
      eventPhase
      : la phase de l'événement en cours. 1 = phase de capture, 2 = phase de bouillonnement sur la cible(bubbling), 3 = bouillonnement sur les ancêtres de la cible, 0 = événement créé artificiellement et pas encore déclenché. Non compatible IE.

  • Propriétés relatives aux événements de type Key Events(événements liés aux touches du clavier):
    • Boolean 
      shiftKey
      : 
      true
      si la touche "shift" a été pressée, 
      false
      sinon.
    • Boolean 
      ctrlKey
      : 
      true
      si la touche "ctrl" a été pressée, 
      false
      sinon.
    • Boolean 
      altKey
      : 
      true
      si la touche "alt" a été pressée, 
      false
      sinon.
    • Boolean 
      metaKey
      : 
      true
      si la touche "Windows" ou "Apple" a été pressée, 
      false
      sinon.
    • Number 
      modifiers
      : remplace les 3 propriétés précédentes pour les navigateurs ne les supportant pas. 1 = "alt", 2 = "ctrl", 4 = "shift", 8 = "cmd"(Mac).
    • Number 
      keyCode
      : correspond au numéro de la touche pressée.
    • Number 
      which
      : correspond au numéro de la touche pressée pour les navigateurs ne supportant pas 
      keyCode
      .
    • Number 
      charCode
      : valeur du caractère unicode associé à la touche. Valable uniquement pour Mozilla.

  • Propriétés relatives aux événements de type Mutation Events(événements liés aux modification du document):
    • Number 
      attrChange
      : type de changement pour l'événement 'DOMAttrModified'. 1 = modification, 2 = addition, 3 = suppression. Non compatible IE.
    • String 
      attrName
      : nom de l'attribut pour l'événement 'DOMAttrModified'. Non compatible IE.
    • String 
      newValue
      : nouvelle valeur de 
      nodeValue
      . Non compatible IE.
    • String 
      prevValue
      : ancienne valeur de 
      nodeValue
      . Non compatible IE.
    • Element 
      relatedNode
      : similaire à 
      relatedTarget
      pour le type mutation events. Non compatible IE.

  • Propriétés relatives aux événements de type Mouse Events(événements liés à la souris):
    • Number 
      button
      : bouton de la souris responsable du clic. 0 ou 1 = bouton gauche, 2 = bouton droit.
    • Number 
      which
      : bouton de la souris responsable du clic pour les navigateurs ne supportant pas 
      button
      . 1 = bouton gauche, 3 = bouton droit.
    • Variant 
      detail
      : informations supplémentaires dépendant de l'événement, par exemple pour un clic, la propriété peut contenir le ombre de clics consécutifs. Non compatible IE.
    • Number 
      pageX
      : position horizontale de la souris par rapport à la page. Non compatible IE.
    • Number 
      pageY
      : position verticale de la souris par rapport à la page. Non compatible IE.
    • Number 
      clientX
      : Idem 
      pageX
      pour IE.
    • Number 
      clientY
      : Idem 
      pageY
      pour IE.
    • Number 
      screenX
      : position horizontale de la souris par rapport à l'écran.
    • Number 
      screenY
      : position verticale de la souris par rapport à l'écran.
    • Element 
      relatedTarget
      : élément significatif pour l'événement. Par exemple l'élément sur lequel la souris était avant de passer sur l'élément actuel pour l'événement 'mouseover'. Non compatible IE.
    • Element 
      fromElement
      : élément sur lequel la souris était avant de passer sur l'élément actuel pour l'événement 'mouseover'. IE uniquement.
    • Element 
      toElement
      : élément que la souris est sur le point de quitter pour l'événement 'mouseout'. IE uniquement.
